## Overview

The Filter module provides CSS-like filter functions for canvas contexts. All functions return filter strings that can be applied to canvas elements.

## Functions

### blur

Applies a Gaussian blur effect to the canvas.

```typescript
function blur(amount: number): string
```

<ParamField path="amount" type="number" required>
  The blur radius in pixels
</ParamField>

**Returns:** A CSS filter string

```typescript
import { createCanvas } from '@napi-rs/canvas';
import { blur } from '@magicyan/canvas';

const canvas = createCanvas(800, 600);
const ctx = canvas.getContext('2d');

ctx.filter = blur(5);
ctx.fillRect(0, 0, 800, 600);
```

### dropShadow

Adds a drop shadow effect to canvas elements.

```typescript
function dropShadow(
  x: number,
  y: number,
  blur: number,
  color?: string | CssColors
): string
```

<ParamField path="x" type="number" required>
  The horizontal offset of the shadow in pixels
</ParamField>

<ParamField path="y" type="number" required>
  The vertical offset of the shadow in pixels
</ParamField>

<ParamField path="blur" type="number" required>
  The blur radius of the shadow in pixels
</ParamField>

<ParamField path="color" type="string | CssColors" default="'black'">
  The color of the shadow (CSS color string or named color)
</ParamField>

**Returns:** A CSS filter string

```typescript
import { createCanvas } from '@napi-rs/canvas';
import { dropShadow } from '@magicyan/canvas';

const canvas = createCanvas(800, 600);
const ctx = canvas.getContext('2d');

// Add a drop shadow with custom color
ctx.filter = dropShadow(5, 5, 10, 'rgba(0, 0, 0, 0.5)');
ctx.fillStyle = 'white';
ctx.fillRect(100, 100, 200, 200);
```

## Combining Filters

You can combine multiple filters by joining them with spaces:

```typescript
import { createCanvas } from '@napi-rs/canvas';
import { blur, brightness, contrast } from '@magicyan/canvas';

const canvas = createCanvas(800, 600);
const ctx = canvas.getContext('2d');

// Combine multiple filters
ctx.filter = [blur(2), brightness(120), contrast(110)].join(' ');
ctx.fillRect(0, 0, 800, 600);
```

## Filter Object

All filter functions are also available as methods on the `Filter` object:

```typescript
import { Filter } from '@magicyan/canvas';

const blurFilter = Filter.blur(5);
const opacityFilter = Filter.opacity(50);
```
