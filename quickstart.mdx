Get up and running with Magicyan in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- Node.js v18 or higher installed
- A Discord bot token ([Create one here](https://discord.com/developers/applications))
- Basic knowledge of JavaScript/TypeScript and Discord.js

## Installation

<Steps>
  <Step title="Create a new project">
    Initialize a new Node.js project:
    
    ```bash
    mkdir my-discord-bot
    cd my-discord-bot
    npm init -y
    ```
  </Step>

  <Step title="Install dependencies">
    Install Magicyan packages and Discord.js:
    
    <CodeGroup>
    ```bash npm
    npm install discord.js @magicyan/core @magicyan/discord
    ```
    ```bash yarn
    yarn add discord.js @magicyan/core @magicyan/discord
    ```
    ```bash pnpm
    pnpm add discord.js @magicyan/core @magicyan/discord
    ```
    </CodeGroup>
  </Step>

  <Step title="Create your bot file">
    Create an `index.js` file with your bot code:
    
    ```javascript index.js
    import { Client, GatewayIntentBits } from 'discord.js';
    import { createEmbed, createRow, createLinkButton } from '@magicyan/discord';
    import { brBuilder, capitalize } from '@magicyan/core';

    const client = new Client({
      intents: [
        GatewayIntentBits.Guilds,
        GatewayIntentBits.GuildMessages,
        GatewayIntentBits.MessageContent
      ]
    });

    client.on('ready', () => {
      console.log(`Bot is ready as ${client.user.tag}`);
    });

    client.on('messageCreate', async (message) => {
      if (message.author.bot) return;
      
      if (message.content === '!hello') {
        const embed = createEmbed({
          color: '#7c7c7c',
          title: capitalize('welcome to magicyan!'),
          description: brBuilder(
            'This is a simple example bot using Magicyan utilities.',
            '',
            'Features:',
            '• Easy embed creation',
            '• String manipulation',
            '• Component builders'
          ),
          footer: { text: 'Powered by Magicyan' },
          timestamp: true
        });

        const row = createRow(
          createLinkButton({
            label: 'Documentation',
            url: 'https://github.com/rinckodev/magicyan'
          })
        );

        await message.reply({ embeds: [embed], components: [row] });
      }
    });

    client.login('YOUR_BOT_TOKEN');
    ```
  </Step>

  <Step title="Run your bot">
    Start your bot:
    
    ```bash
    node index.js
    ```
    
    Your bot should now be online! Send `!hello` in a Discord channel where your bot has access.
  </Step>
</Steps>

## Using Core Utilities

Here are some quick examples of @magicyan/core utilities:

```typescript
import { 
  equalsIgnoreCase, 
  limitText, 
  sleep,
  isDefined 
} from '@magicyan/core';

// Case-insensitive comparison
if (equalsIgnoreCase(input, 'HELLO')) {
  console.log('Match!');
}

// Truncate text
const short = limitText('This is a very long text', 20, '...');
// Result: "This is a very lo..."

// Async sleep
await sleep.seconds(5); // Wait 5 seconds

// Type guard
const values = [1, null, 2, undefined, 3];
const defined = values.filter(isDefined); // [1, 2, 3]
```

## Using Discord Utilities

Explore more Discord.js helpers:

```typescript
import { 
  findChannel, 
  findMember, 
  findCommand,
  commandMention 
} from '@magicyan/discord';

// Find channels with type safety
const channel = findChannel(guild)
  .byName('general', (ch) => ch.isTextBased());

// Find members
const member = findMember(guild)
  .byUsername('rincko');

// Find and mention commands
const cmd = findCommand(client).byName('help');
if (cmd) {
  const mention = commandMention(cmd);
  // Returns: </help:123456789>
}
```

## Building Components

Create interactive Discord components easily:

```typescript
import { 
  createEmbed, 
  createRow, 
  createButton,
  createModal 
} from '@magicyan/discord';

// Create a button row
const buttons = createRow(
  createButton({
    customId: 'confirm',
    label: 'Confirm',
    style: 'Success'
  }),
  createButton({
    customId: 'cancel',
    label: 'Cancel',
    style: 'Danger'
  })
);

// Create a modal
const modal = createModal({
  customId: 'feedback',
  title: 'Send Feedback',
  components: [
    createRow(
      createInput({
        customId: 'message',
        label: 'Your Message',
        style: 'Paragraph',
        required: true
      })
    )
  ]
});
```
