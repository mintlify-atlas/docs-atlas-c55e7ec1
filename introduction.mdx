## What is Magicyan?

Magicyan is a modern, type-safe SDK library consisting of TypeScript utility packages designed specifically for Discord.js bot development. It provides a collection of lightweight, modular utilities that simplify common tasks and enhance your bot development experience.

<Note>
  All Magicyan packages are built with TypeScript and provide full type safety, ensuring a better developer experience with autocompletion and type checking.
</Note>

## Package Ecosystem

Magicyan consists of several specialized packages, each solving specific problems in bot development:

### Core Utilities

<Card title="@magicyan/core" icon="gear" href="/packages/core">

## Who is Magicyan For?

Magicyan is perfect for:

- **Discord Bot Developers** building bots with Discord.js who want to reduce boilerplate and improve productivity
- **TypeScript Developers** who value type safety and want comprehensive type definitions for their Discord projects
- **Teams** who need consistent, maintainable utilities across multiple bot projects
- **Beginners** learning Discord bot development who want intuitive, well-documented helper functions
- **Advanced Developers** who want low-level utilities that don't get in the way of custom implementations

## Design Philosophy

Magicyan follows these core principles:

1. **Type Safety First**: Every function and utility is fully typed with TypeScript
2. **Zero Magic**: Clear, explicit APIs that do exactly what they say
3. **Composable**: Small, focused utilities that work well together
4. **Tree-Shakeable**: Modern ES modules allow bundlers to remove unused code
5. **Framework Agnostic**: Core utilities work anywhere, Discord packages integrate seamlessly with Discord.js

## Requirements

<Info>
  **Node.js**: v18 or higher  
  **Discord.js**: v14.25.1 or higher (for Discord-related packages)  
  **Package Manager**: npm, yarn, or pnpm
</Info>
