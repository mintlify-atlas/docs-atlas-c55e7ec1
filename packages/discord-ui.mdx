The `@magicyan/discord-ui` package provides ready-to-use interactive UI components for Discord bots, including pagination menus, multi-page menus, and confirmation prompts. These components handle all the complexity of button collectors and state management.

## Installation

<CodeGroup>
```bash npm
npm install @magicyan/discord-ui
```

```bash pnpm
pnpm add @magicyan/discord-ui
```

```bash yarn
yarn add @magicyan/discord-ui
```
</CodeGroup>

## Core Exports

### Functions

- `discordUI()` - Configure global defaults for UI components
- `pagination()` - Create paginated embed menus
- `multimenu()` - Create multi-section menus
- `confirm()` - Create confirmation prompts

## Global Configuration

Configure default styles for all UI components:

```typescript
import { discordUI } from '@magicyan/discord-ui';
import { ButtonStyle } from 'discord.js';

// Set defaults once at bot startup
discordUI({
  prompts: {
    confirm: {
      buttons: {
        confirm: {
          label: "Yes",
          emoji: "✅",
          style: ButtonStyle.Success
        },
        cancel: {
          label: "No",
          emoji: "❌",
          style: ButtonStyle.Danger
        }
      }
    }
  },
  menus: {
    pagination: {
      buttons: {
        previous: {
          label: "Previous",
          emoji: "◀️",
          style: ButtonStyle.Primary
        },
        next: {
          label: "Next",
          emoji: "▶️",
          style: ButtonStyle.Primary
        }
      }
    },
    multimenu: {
      viewType: "button", // or "select"
      placeholder: "Select a page..."
    }
  }
});
```

## Pagination Menu

Create navigable paginated content:

### Basic Example

```typescript
import { pagination } from '@magicyan/discord-ui';
import { EmbedBuilder } from 'discord.js';

// Create pages
const pages = [
  new EmbedBuilder()
    .setTitle("Page 1")
    .setDescription("This is the first page"),
  new EmbedBuilder()
    .setTitle("Page 2")
    .setDescription("This is the second page"),
  new EmbedBuilder()
    .setTitle("Page 3")
    .setDescription("This is the third page")
];

// Create pagination
await pagination({
  interaction,
  pages,
  time: 60000, // Collector timeout in ms
  ephemeral: false
});
```

### Advanced Pagination

```typescript
import { pagination } from '@magicyan/discord-ui';

await pagination({
  interaction,
  pages: [
    { embeds: [embed1], content: "Page 1" },
    { embeds: [embed2], content: "Page 2" },
    { embeds: [embed3], content: "Page 3" }
  ],
  
  // Customize buttons for this instance
  buttons: {
    previous: {
      emoji: "⬅️",
      label: "Back"
    },
    next: {
      emoji: "➡️",
      label: "Forward"
    }
  },
  
  // Custom button filter
  filter: (buttonInteraction) => {
    return buttonInteraction.user.id === interaction.user.id;
  },
  
  // Callbacks
  onTimeout: async () => {
    await interaction.editReply({
      content: "Menu timed out",
      components: []
    });
  },
  
  onPageChange: async (page, pageIndex) => {
    console.log(`User navigated to page ${pageIndex + 1}`);
  },
  
  time: 120000 // 2 minutes
});
```

## Confirmation Prompt

Create simple yes/no confirmation dialogs:

### Basic Confirm

```typescript
import { confirm } from '@magicyan/discord-ui';
import { ActionRowBuilder, ButtonBuilder } from 'discord.js';

await confirm({
  // Render the initial message
  render: async (components) => {
    return await interaction.reply({
      content: "Are you sure you want to delete this?",
      components,
      fetchReply: true
    });
  },
  
  // Handle button click
  onClick: async (buttonInteraction, isCancel) => {
    if (isCancel) {
      await buttonInteraction.update({
        content: "Cancelled!",
        components: []
      });
    } else {
      await buttonInteraction.update({
        content: "Deleted successfully!",
        components: []
      });
      // Perform delete action
    }
  },
  
  time: 30000 // 30 seconds
});
```

### Custom Buttons

```typescript
import { ButtonStyle } from 'discord.js';

await confirm({
  buttons: {
    confirm: {
      label: "Accept",
      emoji: "👍",
      style: ButtonStyle.Success
    },
    cancel: {
      label: "Decline",
      emoji: "👎",
      style: ButtonStyle.Danger
    }
  },
  
  render: async (components) => {
    return await interaction.reply({
      content: "Do you accept the terms?",
      components,
      fetchReply: true
    });
  },
  
  onClick: async (buttonInteraction, isCancel) => {
    if (!isCancel) {
      await buttonInteraction.update({
        content: "Terms accepted! ✅",
        components: []
      });
    } else {
      await buttonInteraction.update({
        content: "Terms declined.",
        components: []
      });
    }
  }
});
```

## Real-World Examples

### Shop with Pagination

```typescript
import { pagination } from '@magicyan/discord-ui';

const items = await getShopItems();
const itemsPerPage = 5;

// Create pages from items
const pages = [];
for (let i = 0; i < items.length; i += itemsPerPage) {
  const pageItems = items.slice(i, i + itemsPerPage);
  const embed = new EmbedBuilder()
    .setTitle("Shop")
    .setDescription(pageItems.map((item, idx) => 
      `${i + idx + 1}. ${item.name} - ${item.price} coins`
    ).join('\n'))
    .setFooter({ text: `Page ${Math.floor(i / itemsPerPage) + 1}/${Math.ceil(items.length / itemsPerPage)}` });
  
  pages.push({ embeds: [embed] });
}

await pagination({ interaction, pages, time: 120000 });
```

## Best Practices

<Tip>
  Configure global defaults with `discordUI()` at bot startup to maintain consistent styling across all UI components.
</Tip>

<Note>
  Always include a timeout handler to clean up expired components and prevent cluttered channels.
</Note>

<Warning>
  Remember to use `fetchReply: true` when replying to interactions if you need the message object for the collector.
</Warning>
